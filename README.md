Project4Rivelli:

Description:
This Java program calculates a performance score for a computer based on its GPU clock speed, CPU clock speed, number of CPU cores, monitor resolution, and overclocking capability.
Based on the score, it recommends the optimal graphics quality for gaming or other performance-intensive tasks.

Key Features:
Prompts the user to enter hardware specifications (GPU, CPU, cores, monitor resolution).
Validates user input to ensure realistic hardware values.
Calculates a performance score using a weighted formula.
Provides a recommended graphics quality setting (Ultra, Medium, Low, etc.).
Allows multiple computers to be entered and calculates the average performance score.
